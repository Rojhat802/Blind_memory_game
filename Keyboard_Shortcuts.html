Keyboard Shortcuts

Memory shortcuts (access keys) for Windows are:

The Shortcuts are:
  • Alt + h will take you to card number home page
  • Alt + k will take you to card number keyboard shortcuts page( this one)
  • Alt + Q will take you to card number FAQ page
  • Alt + c will take you to card number contacts page
  • Alt + w will take you to card number WCAG/ARIA page

    The Shortcuts for Playing/selecting game
  • Alt + p will start the game based on your chosen game mode
  • Alt + g will let you play again with the same difficulty(will only promt if you win the game)
  • Alt + c will reset the current game
