WCAG and ARIA

WCAG

WCAG, also known as web content accessibility guidelines, is part of a series of web accessibility guidelines publiched by the main international standards. These standards help websites like this help websites achieve otpimum levels of accessbility to allow as many users as possible to be subject to the website. Many of the principles and guidelines have been closely followed and many features have adapted to follow these guidelines. There is lots more information WCAG and the procedures and guidelines in WCAG, Simple Click this link to learn mor about WCAG.

ARIA

What is ARIA? it stands for Accessible Rich Internet Applications . It is a set of attributes that provide the user with extra content with regards to the layout, content UI etc. This website makes full use of ARIA within the code, which in turn allows the user to have grasp of what i happening on the screen , without actually having to look at it.
